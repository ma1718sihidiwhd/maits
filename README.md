# maits - ITコンサルティング


社内SEとして11年の実務経験を基盤に、SIer、フリーランス、法人経営を経て、  
**PM,PMO,ITSM,GRC** を横断したITコンサルティングを実践しています。  
（**PM経験：約14年、チームメンバー数：3〜10名**）

---

## 🏆 代表実績（領域別）

### **GRC（ガバナンス、リスク、コンプライアンス）**
- ソフトウエア開発業（70名規模）  
  ・セキュリティ対策の検討、情報セキュリティポリシ再整備、組織構築を実施。  
  ・FISCガイドライン対応のアセスメントおよび運用策定を支援。  
- 大手小売業（スーパー、600,000名規模）  
  ・現場整合を重視したITガバナンスガイドラインの見直しを実施。

### **ITSM（ITサービスマネジメント）**
- 輸送用機械器具製造業（10,000名規模）  
  ・翻訳ツールを活用し、海外メンバー含む10名規模のヘルプデスク運用移管を実現。  
- 大手小売業（コンビニエンスストア、8,000名規模）  
  ・運用改善提案および品質管理組織の設置提案を実施。  
- プラスチック製品製造業（1,500名規模）  
  ・ITIL導入により問い合わせ対応ルールを整備し、対応効率を改善。

### **PM/PMO（プロジェクトマネジメント）**
- 公共特殊法人（500名規模）  
  ・設計書レビューを通じて設計品質を向上。  
- 大手航空旅客運送業（13,000名規模）  
  ・Active Directory関連の4プロジェクトをPMとして推進・完遂。  
- 専門技術サービス（高速道路／1,200名規模）  
  ・1か月遅延のプロジェクトを立て直し、完遂。
